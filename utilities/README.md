# Utility Scripts

General-purpose utility scripts and helper tools.

## Available Scripts

### file-organizer.sh / file-organizer.ps1
Organizes files by type, date, or custom criteria

### bulk-rename.sh / bulk-rename.ps1
Renames multiple files based on patterns

### duplicate-finder.sh / duplicate-finder.ps1
Finds duplicate files based on content hash

### system-report.sh / system-report.ps1
Generates comprehensive system report (combines multiple checks)

### text-processor.sh
Common text processing operations

### json-formatter.sh / json-formatter.ps1
Validates and formats JSON files

## Use Cases

### File Management
- Organize downloads folder
- Clean up old files
- Find disk space hogs
- Remove duplicates

### Data Processing
- Format configuration files
- Process log files
- Extract data patterns
- Convert file formats

### System Administration
- Generate reports
- Collect diagnostic info
- Batch operations
- Automation helpers

## Tips

1. **Test First** - Try commands on sample data
2. **Backup Important Files** - Before bulk operations
3. **Use Dry-Run** - Preview changes when available
4. **Document Custom Changes** - Note any modifications
5. **Combine Scripts** - Chain utilities for complex tasks

## Customization

These scripts are designed to be customized:
- Modify parameters
- Adjust thresholds
- Change output formats
- Add custom logic

## Integration

Utilities can be:
- Called from other scripts
- Scheduled with cron/Task Scheduler
- Integrated into workflows
- Used interactively
